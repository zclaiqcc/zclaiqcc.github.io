第一周：基础

目标：第一周是前端基础以及相关知识的补充学习，前端基础主要包括 HTML, CSS, JS，其中以 JS 为主，相关知识有比如 git, markdown 等

学习方法推荐边读文档、边写代码、边做笔记，最后的输出是：一份样例代码、一份学习文档

以下网站根据自己实际情况取阅，中英混杂，仅做推荐，欢迎补充

第一月：框架

Angular

  • 项目使用 Angular（非 AngularJS，自行查阅区别）

  • Ant Design for NG - UI 库

Tasks

  1. Git: 使用 Git 维护一个项目，把联系代码等都托管到 Gitlab 上
  2. Markdown: 使用 markdown 记录文档和笔记
  3. 学习 Angular 使用（官方文档的“教程”以及“核心知识”部分）
  4. 了解 Ant Design UI 库
  5. 清明节假期会留一个小作业，使用 Angular 框架，配合 完成
